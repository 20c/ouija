template

(
  • name
)
JQuery result

Defined in src/twentyc.rest.js:460

Clones a template element by name and returns it

Templates for a widget should be stored within an element inside the widget that has it's .templates css class set

A template is designated a template if it has the data-template attribute set to hold the template name

<div class="widget">
 <div class="templates">
   <div data-template="my_template"></div>
 </div>
</div>

var tmpl = widget.template('my_template')

Parameters:

  • name String

    template name (as specified in data-template)

Returns:

JQuery result:

jquery result holding the cloned template node
